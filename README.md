# 🚀 Smart Project Dashboard (Pro)

A **fully interactive, multi-project project management dashboard** built with **Streamlit & Plotly**, designed for **productivity analysis, team monitoring, and Agile reporting**.

This dashboard allows you to:

- 📥 **Connect directly to Bitrix24** via Inbound Webhook or upload CSV files.
- 📁 **Multi-project management** with dynamic project filtering.
- 👩‍💻 **Role-based categorization**: Testers (Bugs), Frontend, Backend, and other team members.
- 📊 **Interactive visualizations** using Plotly:
  - Task Distribution (Pie Chart)  
  - Tasks per Assignee (Bar Chart)  
  - Bugs per Assignee (Bar Chart)  
  - Burndown Chart (Sprint Progress)  
  - Team Velocity (Weekly Performance)  
  - Velocity per Task Category  
- 📄 **Dynamic data table** with filtering options.
- ⚡ **Real-time analytics** for stakeholders, PMs, and team leads.
- 🔄 **Reusable for any Bitrix24 project** or CSV dataset.

---

## 🚀 Features

- **Interactive UI**: Zoom, hover, and filter visualizations.
- **Multi-project dashboards**: Track multiple projects simultaneously.
- **Role configuration**: Assign testers, frontend, and backend teams dynamically.
- **Burndown & Velocity charts**: Agile-ready project analytics.
- **CSV or Bitrix24 integration**: Flexible data sources.
- **Fully reusable** for any organization or project management workflow.

## ⚡ Quick Start

1. Clone this repo  
```bash
git clone https://github.com/yusufharb/Bitrix24-tasks-analysis-.git
```
2. Install dependencies
```bash
pip install streamlit pandas plotly requests
```
3. run the dashboard 
```bash
streamlit run smart_dashboard_pro.py
```
4. Upload your CSV or connect via Bitrix24 Inbound Webhook

5. Enjoy interactive dashboards with real-time metrics!
